Supported browsers

flexCarousel.js supports the latest versions of all major browsers and platforms. We use Flexbox browser technology to build the carousels, which is supported by these desktop and mobile browsers. For JavaScript, we use jQuery as a dependency (>= 1.10.2) to which you can view the browser support here.

Our compiled JavaScript uses ECMAScript 6, we use this to determine what browsers can support our jQuery plugin which you can view in the tables below.

Internet Explorer

flexCarousel.js has been developed without Internet Explorer in mind, we cannot guarantee that carousels will display or work correctly in any version of this browser. IE10 is only partially supported.
